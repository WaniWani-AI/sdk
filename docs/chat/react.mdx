---

For React apps where you want full control over placement, theming, and lifecycle, import the chat widget directly from `@waniwani/sdk/chat`.

<Note>
**Platform feature.** Requires a WaniWani API key (project ID). [Get one at app.waniwani.ai](https://app.waniwani.ai) or read more about the [Platform](/platform/overview).
</Note>

## Install

```bash
bun add @waniwani/sdk
```

Add the stylesheet to your app once (e.g. in `_app.tsx` or your root layout):

```ts
import "@waniwani/sdk/chat/styles.css";
```

## Basic usage

```tsx
import { ChatWidget } from "@waniwani/sdk/chat";

export default function App() {
  return (
    <ChatWidget
      projectId="wwp_..."
      theme="auto"
      placement="bottom-right"
    />
  );
}
```

`<ChatWidget>` renders a floating bubble + panel, scoped to its own Shadow DOM so your global styles don't leak in (and ours don't leak out).

## Layout variants

For more control over placement, use one of the layout primitives:

```tsx
import { ChatCard, ChatBar, ChatEmbed } from "@waniwani/sdk/chat";

// Pinned card with a fixed size
<ChatCard projectId="wwp_..." className="w-[400px] h-[600px]" />

// Bottom-of-page bar
<ChatBar projectId="wwp_..." />

// Inline embedded chat (fills its container)
<ChatEmbed projectId="wwp_..." />
```

## Theming

The chat exposes a small set of CSS variables for branding:

```tsx
import { ChatWidget, mergeTheme, DEFAULT_THEME } from "@waniwani/sdk/chat";

const theme = mergeTheme(DEFAULT_THEME, {
  primary: "#03d916",
  background: "#fffaf0",
});

<ChatWidget projectId="wwp_..." theme={theme} />
```

## How it works

The widget talks directly to `app.waniwani.ai/api/chat` from the browser. No server-side handler required on your side.
