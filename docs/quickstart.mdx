## Option A: Managed project

WaniWani provisions a GitHub repo, deploys to Vercel, and auto-tunnels local dev so ChatGPT and Claude can reach your server instantly.

<Steps>
  <Step title="Create a managed project">
    1. Open [app.waniwani.ai/mcp](https://app.waniwani.ai/mcp) and sign in.
    2. Click **Create project** and choose **Managed** ("We host your MCP").
    3. WaniWani creates a GitHub repo and Vercel project for you. Two environments are provisioned automatically: **Staging** and **Production**.
    4. Clone the repo locally.
  </Step>

  <Step title="Install and run">
    <Note>Managed projects use **Bun** as the package manager.</Note>

    ```bash
    bun install
    bun dev
    ```

    `bun dev` starts a Cloudflare tunnel automatically. Your local server is now reachable from ChatGPT and Claude — no ngrok, no port forwarding.
  </Step>

  <Step title="Open the playground">
    The dev server prints a tunnel URL. Open it in your browser to access the built-in chat playground, or connect from ChatGPT/Claude using the tunnel URL + `/mcp`.
  </Step>
</Steps>

## Option B: External project

You already have an MCP server and want to add WaniWani to it.

<Steps>
  <Step title="Install the SDK">
    <CodeGroup>
      ```bash bun
      bun add @waniwani/sdk
      ```

      ```bash pnpm
      pnpm add @waniwani/sdk
      ```

      ```bash npm
      npm install @waniwani/sdk
      ```
    </CodeGroup>
  </Step>

  <Step title="Create an external project and get an API key">
    1. Open [app.waniwani.ai/mcp](https://app.waniwani.ai/mcp) and sign in.
    2. Click **Create project** and choose **External** ("You host your MCP").
    3. Set your server URL for the production environment.
    4. Copy the API key (`wwk_...`) — it is shown only once.

    ```bash .env
    WANIWANI_API_KEY=wwk_...
    ```
  </Step>

  <Step title="Wrap your server">
    Call `withWaniwani(server)` once. Every tool call now emits a `tool.called` event to your dashboard.

    ```ts server/src/index.ts
    import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
    import { StreamableHTTPServerTransport } from "@modelcontextprotocol/sdk/server/streamableHttp.js";
    import { withWaniwani } from "@waniwani/sdk/mcp";
    import "dotenv/config";

    const server = new McpServer({ name: "my-mcp", version: "0.0.1" });

    server.registerTool(/* ... */);

    withWaniwani(server);

    const transport = new StreamableHTTPServerTransport({ sessionIdGenerator: undefined });
    await server.connect(transport);
    ```
  </Step>

  <Step title="Verify">
    Trigger any tool call, then check the [WaniWani dashboard](https://app.waniwani.ai). The event should appear within seconds.

    If nothing shows up, confirm `WANIWANI_API_KEY` is set and look for `[waniwani]` lines in your server logs.
  </Step>
</Steps>
